## 🎯 À propos

Cette version est une **amélioration visuelle et interactive** du site BeninBouge.

⚠️ Aucun changement backend ou logique métier n’a été effectué.  
Cette mise à jour concerne uniquement :

- 🎨 L’interface utilisateur (UI)
- 💫 L’expérience utilisateur (UX)
- ⚡ Les animations et transitions
- 📱 L’optimisation responsive

---

## ✨ Améliorations apportées

- Animations d’apparition au scroll
- Transitions fluides entre sections
- Micro-interactions modernes
- Effets dynamiques sur boutons et cartes
- Amélioration de la hiérarchie visuelle
- Expérience plus immersive et professionnelle

---

## 🛠️ Technologies d’animation utilisées

- 🎬 Anime.js  
- 🎞️ GSAP (GreenSock Animation Platform)  
- 🌊 AOS (Animate On Scroll)  

---

## 🎨 Objectif de cette amélioration

- Moderniser l’image du site
- Rendre l’expérience plus dynamique
- Améliorer l’engagement utilisateur
- Donner un aspect plus premium et professionnel

---

## ⚙️ Stack principale

- Laravel
- React + Inertia.js
- Tailwind CSS
- Anime.js
- GSAP
- AOS
